# FlexNetOS Claude Plugins

A [Claude Code plugin marketplace](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/plugins) by FlexNetOS.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [meta](https://github.com/FlexNetOS/meta) | Multi-repo workspace management — skills, hooks, and MCP server. |
| [gitkb](https://github.com/FlexNetOS/claude-plugins) | Knowledge base and code intelligence for AI-native development. |

## Usage

```bash
# Add this marketplace
claude plugin marketplace add FlexNetOS/claude-plugins

# Install a plugin
claude plugin install meta@FlexNetOS
claude plugin install gitkb@FlexNetOS
```
